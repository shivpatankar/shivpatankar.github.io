Student Course Registration System

This is a student portal app that provides students access to university's programs and other learning related materials.

Highlights

  • The application is built using Jersey RESTful web service framework
  • The application is connected to Amazon's DynamoDB to store data
  • The application uses AWS SNS to send notifications to students
  • Application endpoints are tested on Postman
  • Application is hosted on Amazon's Elastic beanstalk

Tools & Technologies: Java EE, Jersey framework, Postman, Elastic Beanstalk, Amazon S3, RESTful Services, DynamoDB
